# NAF | Tikinti Texnikası İcarəsi & Admin Dashboard

Modern və estetik monoxrom dizayna malik, tam dinamik tikinti texnikası icarəsi platforması. Next.js, Supabase və Framer Motion texnologiyaları ilə mühəndislik səviyyəsində qurulub.

## 🚀 Texnologiyalar

- **Frontend:** Next.js (App Router), React 19, Tailwind CSS
- **Animasiya:** Framer Motion (Axıcı keçidlər və parallaks effektlər)
- **Backend/DB:** Supabase (Real-time data sinxronizasiyası)
- **Tipoqrafiya:** Serif & Sans-serif (Playfair Display / Inter)

## ✨ Əsas Xüsusiyyətlər

- **Dinamik Avadanlıq Parkı:** Bütün texnikalar və məlumatlar Supabase DB-dən real-vaxtda çəkilir.
- **Haqqımızda Bölməsi:** Admin panelindən idarə olunan dinamik mətn sistemi.
- **Professional Admin Paneli:** `/admin` ünvanında yerləşən, monoxrom dizaynlı tam CRUD (Yarat, Oxu, Yenilə, Sil) sistemi.
- **Sifariş İdarəetməsi:** Gələn sifarişlərin admin tərəfdən statusunun (Pending, Accepted, Completed) dəyişdirilməsi.
- **Blueprint Estetikası:** Bütün sayt boyu mühəndislik çertyojlarını xatırladan incə arxa fon detalları.

## 🛠️ Quraşdırılma

1. Layihəni klonlayın:
```bash
git clone https://github.com/UmudvarKhalisli/tikinti-texnikasi.git
```

2. Dependency-ləri yükləyin:
```bash
npm install
```

3. `.env.local` faylını yaradın və Supabase açarlarınızı daxil edin:
```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
```

4. Dev server-i başladın:
```bash
npm run dev
```
