# NAF Texnika SEO Master Plan (Azerbaijan Market)

## Purpose
This document is a practical SEO implementation brief for **naftexnika.az**. It is written so a coding assistant like Copilot can use it to implement on-page SEO, technical SEO, structured data, content planning, and metadata updates across the project.

---

## Project Context
- Brand: **NAF Texnika**
- Main domain: **https://naftexnika.az**
- Redirect domain: **https://www.naftexnika.az** -> **https://naftexnika.az**
- Market: **Azerbaijan**
- Language priority: **Azerbaijani**
- Business type: **Construction machinery rental / heavy equipment rental**
- Main service areas: **Baku, Absheron, Sumqayit, Xirdalan**
- Primary commercial intent: generate inbound leads via call, WhatsApp, and quote forms

---

## Core SEO Strategy

### Main goals
1. Rank for commercial keywords related to construction machinery rental in Azerbaijan
2. Build landing pages for each machinery category
3. Build local landing pages for important regions
4. Support rankings with blog articles and FAQ content
5. Strengthen technical SEO, metadata, sitemap, robots, canonical, internal links, and schema

### Important rules
- Use **naftexnika.az** as the canonical domain everywhere
- Do not keep old references to **naf-company.vercel.app**
- Every indexable page must have:
  - unique title
  - unique meta description
  - one H1
  - useful body content
  - internal links
  - canonical URL
  - Open Graph tags
  - structured data where relevant
